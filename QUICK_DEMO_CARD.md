# 🎯 Quick Demo Card - Snacks Hub

## 🚀 Start Server
```bash
node server.js
```
Open: http://localhost:3000

---

## ⚡ 5-Minute Demo Script

### 1️⃣ Homepage (30s)
- Show hero section
- Click language buttons: 🇬🇧 EN → 🇮🇳 HI → 🇪🇸 ES
- Toggle dark mode: 🌙 ↔ ☀️

### 2️⃣ Search & Browse (30s)
- Type "burger" in search → see autocomplete with images
- Click category filters
- Scroll through menu items

### 3️⃣ Customize Meal (1m)
- Click **Customize** button (🎛️) on any item
- Select "Large" size → price increases
- Add "Extra Cheese" + "Bacon" → price updates
- Choose "Extra Spicy"
- Type special instructions
- Click "Add to Cart"

### 4️⃣ Share Item (30s)
- Click **Share** button (📤) on any item
- Show WhatsApp, Facebook, Twitter, Telegram options
- Click "Copy Link" → toast notification

### 5️⃣ Cart & Checkout (30s)
- Open cart sidebar
- Show customized item with all options
- Adjust quantities with +/- buttons
- Click "Proceed to Checkout"

### 6️⃣ Group Order (45s)
- Login first (if not logged in)
- Click **"👥 Group Order"** button in nav
- Show shareable link
- Click "Copy" button
- Explain: "Friends can open this link and add items together"

### 7️⃣ Analytics Dashboard (1m)
- Click **"📊 Analytics"** button in nav
- Point out:
  - Total Orders: 📦
  - Total Revenue: 💰
  - Avg Order Value: 📈
  - Top 5 Popular Items: 🔥
  - Orders by Status: colored bar chart
  - 7-Day Revenue: bar chart with hover

### 8️⃣ Order History (30s)
- Go to Dashboard
- Show order history
- Click **"Reorder"** button → all items added to cart instantly
