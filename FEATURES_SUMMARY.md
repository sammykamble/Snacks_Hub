## 🐛 Troubleshooting:

### Features not showing?
1. Make sure server is running: `node server.js`
2. Clear browser cache (Ctrl + Shift + Delete)
3. Hard refresh (Ctrl + F5)
4. Check browser console for errors (F12)

### Dark mode button not visible?
- Look in bottom-right corner
- Scroll down if needed
- Check if CSS file loaded (DevTools → Network)

### Search not working?
- Make sure you're typing in the navigation search bar
- Wait 300ms for results to appear
- Check if menu items are loaded
